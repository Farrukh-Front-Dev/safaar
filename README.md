# Agoda Monorepo

This repository contains the Agoda frontend and backend apps in one npm workspace.

## Apps

- `apps/frontend` - Next.js frontend
- `apps/backend` - NestJS backend

## Setup

```bash
npm install
```

## Development

```bash
# Run frontend and backend together
npm run dev

# Run only the backend
npm run dev:backend

# Run only the frontend
npm run dev:frontend
```

## Build

```bash
npm run build
```

## Tests

```bash
npm run test
npm run test:backend
```
